# Fase 5 – Evaluación Final POA

## Descripción del proyecto

Este proyecto fue desarrollado como actividad final del curso **Fundamentos de Programación** de la Universidad Nacional Abierta y a Distancia (UNAD).  

La aplicación tiene como objetivo evaluar el nivel de compromiso de diferentes clientes a partir de la duración de sus sesiones y la cantidad de clics realizados durante la navegación.

El sistema clasifica cada sesión en tres niveles:

- Alto
- Medio
- Bajo

La clasificación se realiza utilizando estructuras condicionales, matrices y funciones básicas de programación.

---

## Objetivo

Diseñar una solución que permita analizar registros de sesiones de clientes y generar una clasificación automática según ciertos criterios establecidos.
